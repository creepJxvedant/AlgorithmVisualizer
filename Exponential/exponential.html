Exponential Search & Binary Search Code

function exponentialSearch(arr, value) {
    if (arr[0] === value) return 0;
    let bound = 1;
    while (bound < arr.length && arr[bound] < value) {
        bound *= 2;
    }
    return binarySearch(arr, value, bound / 2, Math.min(bound, arr.length - 1));
}

function binarySearch(arr, value, left, right) {
    while (left <= right) {
        let mid = Math.floor((left + right) / 2);
        if (arr[mid] === value) {
            return mid;
        } else if (arr[mid] < value) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    return -1;
}

About Exponential Search

Exponential Search is an algorithm that combines the strengths of Binary Search with a dynamic approach to determine the search range. It is highly efficient for large, sorted datasets.

The algorithm begins by finding the range where the target element lies by repeatedly doubling the index (exponentially growing bounds). Once the range is determined, it applies Binary Search within that range to locate the target.

Key Features:

  • Efficient: Time complexity of O(log N) for searching in sorted arrays.
  • Range Discovery: Dynamically finds the subarray containing the target.
  • Divide-and-Conquer: Utilizes Binary Search for precise element location.

Advantages:

  • Faster than Binary Search for large datasets with unknown boundaries.
  • Minimal space requirements as it operates in-place.

Disadvantages:

  • Requires a sorted dataset, similar to Binary Search.
  • Performance may degrade if the target element lies at the beginning or far outside the dataset.

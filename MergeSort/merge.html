About Merge Sort

Merge Sort is a divide-and-conquer algorithm that splits the list into smaller sublists, sorts each sublist, and then merges them back together. The algorithm repeatedly divides the array in half until each sublist contains a single element and then merges them back together in sorted order.

The merge process ensures that the resulting list is sorted because it repeatedly merges smaller sorted lists into larger ones.

Key Features:

  • Divide and Conquer: The list is divided into smaller parts and solved recursively.
  • Stable Sorting: Maintains the order of equal elements.
  • Efficient for large datasets: Has an optimal time complexity.

Advantages:

  • Time complexity of O(N log N) in the best, worst, and average cases.
  • Stable and efficient for large datasets.
  • Works well on linked lists.

Disadvantages:

  • Requires additional space for merging, which is O(N).
  • Not suitable for smaller datasets where simpler algorithms like QuickSort or Insertion Sort may perform better.
